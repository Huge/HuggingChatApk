# HuggingChatApk
HuggingChatApk is a simple webView wrapper for huggingChat.
[<img src="https://fdroid.gitlab.io/artwork/badge/get-it-on.png"
alt="Get it on F-Droid"
height="60" style="margin: 0;">](https://f-droid.org/packages/org.woheller69.hugassist/)

It blocks all URLs that are not essential for functioning.
Please use an external browser for initial sign-up.
